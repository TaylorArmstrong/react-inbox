## A Mail Inbox
Made with React.js


## Front End Setup
git clone https://github.com/TaylorArmstrong/react-inbox.git 
cd react-inbox
npm i
npm start
Open [http://localhost:3000](http://localhost:3000) to view it in the browser.

## Back End Setup
git clone https://github.com/gSchool/collective-api.git
cd collective-api
npm install
npm start
